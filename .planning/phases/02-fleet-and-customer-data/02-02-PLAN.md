**3. Create `apps/api/src/vehicles/vehicles.service.ts`:**
Inject PrismaService and StorageService.

Methods:
- `create(dto: CreateVehicleDto)` — Create vehicle with nested insurance/inspection using Prisma `create({ data: { ..., insurance: dto.insurance ? { create: dto.insurance } : undefined, inspection: dto.inspection ? { create: dto.inspection } : undefined }, include: { insurance: true, inspection: true, documents: true } })`.
- `findAll(includeArchived = false)` — Default filter: `{ isArchived: false }`. When includeArchived=true, no filter. Always include relations.
- `findOne(id: string)` — `findUniqueOrThrow({ where: { id }, include: { insurance: true, inspection: true, documents: true } })`. Throw NotFoundException if not found.
- `update(id: string, dto: UpdateVehicleDto)` — Fetch existing first, build oldValues diff (comparing each changed field: `{ field: { old: existing[field], new: dto[field] } }`). Handle nested insurance/inspection updates with upsert. Return `{ oldValues, vehicle }`.
  - **Status transition validation:** If dto.status is RENTED or RESERVED, throw BadRequestException("Status RENTED/RESERVED is managed by rental lifecycle"). If current status is RETIRED and dto.status is AVAILABLE/RESERVED/RENTED, throw BadRequestException("Cannot reactivate a retired vehicle").
- `archive(id: string)` — Set `isArchived: true` and `status: RETIRED`.
- `uploadDocument(vehicleId: string, file: Express.Multer.File, label: string)` — Upload to MinIO with key `vehicles/{vehicleId}/documents/{label}-{uuid}.{ext}`. Create VehicleDocument record. Return document record.
- `uploadPhoto(vehicleId: string, file: Express.Multer.File)` — Upload to MinIO with key `vehicles/{vehicleId}/photo.{ext}`. Update vehicle `photoKey`. Return presigned URL.
- `toDto(vehicle: Vehicle & relations)` — Convert Prisma entity to VehicleDto. For photoKey and documentKeys, generate presigned download URLs using StorageService.

**2. Add import endpoint to VehiclesController:**
```typescript
@Post('import')
@Roles(UserRole.ADMIN)
@UseInterceptors(FileInterceptor('file', {
  storage: memoryStorage(),
  limits: { fileSize: 5 * 1024 * 1024 },
  fileFilter: (req, file, cb) => {
    const allowed = [
      'text/csv',
      'application/vnd.ms-excel',
      'application/vnd.openxmlformats-officedocument.spreadsheetml.sheet',
    ];
    cb(null, allowed.includes(file.mimetype) || file.originalname.match(/\.(csv|xls|xlsx)$/i) !== null);
  },
}))
async importFleet(@UploadedFile() file: Express.Multer.File) {
  return this.vehiclesService.importFleet(file);
}
```
