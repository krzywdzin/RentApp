2. Create `apps/api/src/portal/strategies/portal-jwt.strategy.ts`:

Separate passport JWT strategy named `'portal-jwt'` (NOT the default `'jwt'`).
```typescript
import { Injectable } from '@nestjs/common';
import { PassportStrategy } from '@nestjs/passport';
import { ExtractJwt, Strategy } from 'passport-jwt';
import { ConfigService } from '@nestjs/config';

@Injectable()
export class PortalJwtStrategy extends PassportStrategy(Strategy, 'portal-jwt') {
  constructor(config: ConfigService) {
    super({
      jwtFromRequest: ExtractJwt.fromAuthBearerToken(),
      secretOrKey: config.get('JWT_SECRET'),
    });
  }

  async validate(payload: any) {
    // Only accept portal-type tokens
    if (payload.type !== 'portal') return null;
    return { customerId: payload.sub, type: 'portal' };
  }
}
```

4. Create `apps/api/src/portal/portal.service.ts`:

Injectable service with PrismaService, JwtService, StorageService, ConfigService injected.

Methods:
- `generatePortalToken(customerId: string)`: Generate raw token via `crypto.randomBytes(32).toString('base64url')`, hash with argon2, update customer with `portalToken` (hashed) and `portalTokenExpiresAt` (30 days from now). Return raw token. Build portal URL: `${APP_URL}/portal?token=${rawToken}&cid=${customerId}`.
- `exchangeToken(customerId: string, rawToken: string)`: Find customer by ID. If no portalToken or portalTokenExpiresAt, throw UnauthorizedException. If expired, throw UnauthorizedException with message 'Link wygasl. Skontaktuj sie z wypozyczalnia.' Verify token with argon2.verify. If invalid, throw UnauthorizedException. Sign JWT with `{ sub: customerId, type: 'portal' }` and expiresIn '24h'. Return { accessToken }.
- `getRentals(customerId: string)`: Find all rentals for customer, include vehicle and contracts. Map to PortalRentalDto[]. For each rental, find latest SIGNED contract. If contract has pdfKey, generate presigned URL via StorageService.getPresignedUrl (1 hour expiry).
- `getRentalDetail(customerId: string, rentalId: string)`: Find rental with vehicle + contracts include. Verify rental.customerId matches the authenticated customerId (throw NotFoundException if not). Return PortalRentalDto with contract PDF URL and return inspection summary (returnData + returnMileage).

Important: No PII displayed in portal responses per RODO minimization (customer knows their own data). Only return firstName + lastName for header display.
