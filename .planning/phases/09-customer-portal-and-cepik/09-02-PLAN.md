4. Create `apps/api/src/cepik/cepik.controller.ts`:

Two endpoints:
- `POST /cepik/verify` -- `@Roles(UserRole.EMPLOYEE, UserRole.ADMIN)` -- accepts VerifyLicenseDto, returns CepikVerificationDto. Uses @CurrentUser() for checkedById. Returns `__audit` metadata with `{ entityType: 'CepikVerification', entityId, action: 'CREATE' }`.
- `POST /cepik/verify/:id/override` -- `@Roles(UserRole.ADMIN)` -- accepts OverrideCepikDto, returns updated CepikVerificationDto. Returns `__audit` metadata with `{ entityType: 'CepikVerification', entityId, action: 'UPDATE', changes: { status: { old: 'FAILED', new: 'OVERRIDDEN' }, overrideReason: { old: null, new: reason } } }`.
- `GET /cepik/verify/rental/:rentalId` -- `@Roles(UserRole.EMPLOYEE, UserRole.ADMIN)` -- returns latest verification for rental (or 404).
