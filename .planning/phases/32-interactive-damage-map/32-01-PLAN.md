API endpoints (already implemented in apps/api/src/photos/):
- POST /walkthroughs — body: { rentalId: string, type: 'HANDOVER' | 'RETURN' } — creates walkthrough, returns { id, ... }

- POST /damage-reports/:walkthroughId/pins — body: DamagePin fields — adds single pin
- POST /damage-reports/:walkthroughId/no-damage — confirms no damage
- GET /damage-reports/walkthrough/:walkthroughId — gets damage report
