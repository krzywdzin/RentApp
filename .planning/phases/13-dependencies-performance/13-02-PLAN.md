**B) Add filtered rentals hook (PERF-02):**

2. In `use-rentals.ts`, update `useRentals` to accept an optional filters object:
```typescript
interface RentalFilters {
  status?: RentalStatus;
  customerId?: string;
  vehicleId?: string;
}

export function useRentals(filters?: RentalFilters) {
  const params = new URLSearchParams();
  if (filters?.status) params.set('status', filters.status);
  if (filters?.customerId) params.set('customerId', filters.customerId);
  if (filters?.vehicleId) params.set('vehicleId', filters.vehicleId);
  const query = params.toString();
  return useQuery({
    queryKey: rentalKeys.list(filters),
    queryFn: () => apiClient<RentalDto[]>(`/rentals${query ? `?${query}` : ''}`),
  });
}
```
This is backward-compatible -- existing callers passing no args or `useRentals()` still fetch all rentals. Callers passing a RentalStatus directly need updating (check for any `useRentals(someStatus)` calls and change to `useRentals({ status: someStatus })`).
