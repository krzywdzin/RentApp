    3. **device_id cookie refresh** (app/api/auth/refresh/route.ts):
       - Find where `access_token` and `refresh_token` cookies are set in the response.
       - Read the `device_id` cookie from the incoming request: `const deviceId = request.cookies.get('device_id')?.value`.
       - Re-set the `device_id` cookie with the same value and refreshed maxAge (86400 = 24 hours):
         ```

           response.cookies.set('device_id', deviceId, {
             httpOnly: true,
             secure: process.env.NODE_ENV === 'production',
             sameSite: 'lax',
             maxAge: 86400,
             path: '/',
           });
         }
         ```

    4. **Portal auth via React Query** (use-portal-auth.ts):
       - Replace the manual `useState` + `useEffect` + `checkAuth()` pattern with a `useQuery` hook:
         ```
         const { data: user, isLoading, isError } = useQuery({
           queryKey: ['portal', 'auth'],
           queryFn: async () => {
             const res = await fetch('/api/portal/me');
             if (!res.ok) throw new Error('Not authenticated');
             return res.json();
           },
           retry: false,
           staleTime: 5 * 60 * 1000, // 5 minutes
         });
         ```
       - Keep `exchangeToken` and `logout` as separate functions that call `queryClient.invalidateQueries({ queryKey: ['portal', 'auth'] })` after success.
       - Return `{ user, isAuthenticated: !!user, isLoading, exchangeToken, logout }`.
       - This ensures portal auth state is fetched once and shared across all portal pages.

    5. **Sidebar localStorage flash fix** (sidebar.tsx):
       - Find the `useState` for collapsed state and the `useEffect` that reads localStorage.
       - Replace with a useState lazy initializer:
         ```
         const [collapsed, setCollapsed] = useState(() => {
           if (typeof window === 'undefined') return false;
           return localStorage.getItem('sidebar-collapsed') === 'true';
         });
         ```
       - Remove the `useEffect` that reads localStorage on mount.
       - Keep the `useEffect` that writes to localStorage on change (or use the setter directly).
