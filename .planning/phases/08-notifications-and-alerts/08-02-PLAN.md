**Deduplication helper:**
```typescript
private async isDuplicate(type: string, relatedEntityId: string, scheduledDate: Date): Promise<boolean> {
  const startOfDay = new Date(scheduledDate);
  startOfDay.setHours(0, 0, 0, 0);
  const endOfDay = new Date(scheduledDate);
  endOfDay.setHours(23, 59, 59, 999);

  const existing = await this.prisma.notification.findFirst({
    where: {
      type,
      relatedEntityId,
      scheduledFor: { gte: startOfDay, lte: endOfDay },
      status: { in: ['PENDING', 'SENDING', 'SENT'] },
    },
  });
  return !!existing;
}
```

**SMS notification methods:**

  5. Generate message using `returnReminderSms({ returnDate, returnTime, companyPhone })`
  6. Add job to smsQueue

**7. AlertScannerService** (`apps/api/src/notifications/cron/alert-scanner.service.ts`):

```typescript
@Injectable()
export class AlertScannerService {
  private readonly logger = new Logger(AlertScannerService.name);
  constructor(
    private prisma: PrismaService,
    private notificationsService: NotificationsService,
    private alertConfigService: AlertConfigService,
  ) {}

  @Cron('0 8 * * *') // Daily at 8:00 AM
  async scanAlerts() {
    this.logger.log('Starting daily alert scan');
    await this.scanReturnReminders();
    await this.scanOverdueRentals();
    await this.scanInsuranceExpiry();
    await this.scanInspectionExpiry();
    this.logger.log('Daily alert scan complete');
  }
}
```

**scanReturnReminders():**
1. Check AlertConfig for RETURN_REMINDER -- if not enabled, return
2. Get leadTimeDays from config (default 1)
3. Calculate tomorrow in Europe/Warsaw timezone:
   ```typescript
   const now = new Date();
   // Use Intl.DateTimeFormat to get Warsaw date boundaries
   const warsawFormatter = new Intl.DateTimeFormat('en-US', { timeZone: 'Europe/Warsaw', year: 'numeric', month: '2-digit', day: '2-digit' });
   // Calculate start/end of "tomorrow" in Warsaw time
   const warsawNow = new Date(warsawFormatter.format(now));
   const targetDate = new Date(warsawNow);
   targetDate.setDate(targetDate.getDate() + leadTimeDays);
   const startOfTarget = new Date(targetDate.setHours(0, 0, 0, 0));
   const endOfTarget = new Date(targetDate.setHours(23, 59, 59, 999));
   ```
4. Query rentals: `endDate BETWEEN startOfTarget AND endOfTarget, status IN ['ACTIVE', 'EXTENDED']`, include customer and vehicle
5. For each rental: call `notificationsService.enqueueReturnReminder(rental)`

**scanOverdueRentals():**
1. Check AlertConfig for OVERDUE -- if not enabled, return
2. Get maxRepeat from config (default 7)
3. Query rentals: `endDate < now, status IN ['ACTIVE', 'EXTENDED']`, include customer and vehicle
4. For each rental:
   a. Count existing OVERDUE notifications for this rental: `this.prisma.notification.count({ where: { type: 'OVERDUE', relatedEntityId: rental.id, status: { in: ['SENT', 'PENDING', 'SENDING'] } } })`
   b. If count >= maxRepeat, skip
   c. Call `notificationsService.enqueueOverdueAlert(rental, count)`

**scanInsuranceExpiry():**
1. Check AlertConfig for INSURANCE_EXPIRY -- if not enabled, return
2. Get leadTimeDays from config (default 30)
3. Calculate target dates: today + 30 days and today + 7 days (two-stage)
4. Query: `vehicleInsurance WHERE expiryDate BETWEEN startOfTarget AND endOfTarget`, include vehicle
5. For 30-day matches: call `notificationsService.enqueueExpiryAlert(vehicleId, 'INSURANCE_EXPIRY', vehicle, expiryDate, 30)`
6. For 7-day matches: call `notificationsService.enqueueExpiryAlert(vehicleId, 'INSURANCE_EXPIRY', vehicle, expiryDate, 7)`

**scanInspectionExpiry():**
Same pattern as insurance but query `vehicleInspection` table.

```typescript
function formatDateWarsaw(date: Date | string): { date: string; time: string } {
  const d = new Date(date);
  const formatted = d.toLocaleString('pl-PL', {
    timeZone: 'Europe/Warsaw',
    day: '2-digit',
    month: '2-digit',
    year: 'numeric',
    hour: '2-digit',
    minute: '2-digit',
  });
  // pl-PL format: "24.03.2026, 14:00"
  const [datePart, timePart] = formatted.split(', ');
  return { date: datePart, time: timePart };
}
```
