Audit API endpoint (from apps/api/src/audit/audit.controller.ts):
```
GET /audit — @Roles(ADMIN) — query params:
  entityType?: string (e.g., 'Vehicle', 'Customer', 'Rental', 'Contract')
  entityId?: string (UUID)
  actorId?: string (UUID — employee who performed the action)
  limit?: number (1-100, default pagination)
  offset?: number (default 0)
Returns: array of audit log entries
```

Audit log entry shape (from AuditService, stored in Prisma AuditLog model):
```typescript
interface AuditLogEntry {
  id: string;
  action: string;       // 'CREATE' | 'UPDATE' | 'DELETE'
  entityType: string;   // 'Vehicle' | 'Customer' | 'Rental' | 'Contract'
  entityId: string;
  actorId: string;
  changes: Record<string, { old: unknown; new: unknown }> | null;  // field-level diffs
  metadata: Record<string, unknown> | null;
  createdAt: string;
}
// Note: encrypted fields show { old: '[ENCRYPTED]', new: '[ENCRYPTED]' } in changes
```

     - Sub-table headers: "Pole" | "Stara wartosc" | "Nowa wartosc"

     - Use shadcn collapsible or simple state toggle for expansion
