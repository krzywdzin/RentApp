```yaml
- name: Typecheck Mobile

```

b) Test Mobile:
```yaml
- name: Test Mobile
  run: pnpm --filter @rentapp/mobile test
```

c) Run E2E migrations + tests (needs Redis + Postgres, already available from services):
```yaml
- name: Run migrations for E2E
  run: pnpm --filter @rentapp/api exec prisma migrate deploy

- name: E2E Tests
  run: pnpm --filter @rentapp/api test:e2e
```

4. Change the API test step from `test:no-coverage` to `test` (which runs jest --coverage and enforces the threshold):
```yaml
- name: Test API
  run: pnpm --filter @rentapp/api test
```

5. Change the Web test step to include coverage:
```yaml
- name: Test Web
  run: pnpm --filter web test -- --coverage
```

6. Increase timeout-minutes from 10 to 15 (E2E adds time).
