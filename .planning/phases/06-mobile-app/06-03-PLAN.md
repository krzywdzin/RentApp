    3. **src/api/customers.api.ts**: API functions:
       - searchCustomers(query: string): GET /customers?search={query}
       - getCustomer(id: string): GET /customers/:id
       - createCustomer(data: CreateCustomerInput): POST /customers
       Typed with CustomerDto from @rentapp/shared

    4. **src/hooks/use-rentals.ts**: TanStack Query hooks following RESEARCH.md pattern:
       - rentalKeys factory: all, list(filters), detail(id)
       - useRentals(filters?: { status?: string }): useQuery, returns rental list
       - useRental(id: string): useQuery, returns single rental with relations
       - useCreateRental(): useMutation, invalidates rentalKeys.all on success
       - useReturnRental(): useMutation accepting { id, data }, invalidates rentalKeys.all on success
       - useExtendRental(): useMutation, invalidates rentalKeys.all on success

    5. **src/hooks/use-vehicles.ts**: TanStack Query hooks:
       - vehicleKeys factory: all, list(filters), detail(id)
       - useVehicles(filters?: { status?: string }): useQuery
       - useVehicle(id: string): useQuery

    6. **src/hooks/use-customers.ts**: TanStack Query hooks:
       - customerKeys factory: all, search(query), detail(id)
       - useCustomerSearch(query: string): useQuery, enabled when query.length >= 2, keeps previous data during typing
       - useCustomer(id: string): useQuery
       - useCreateCustomer(): useMutation, invalidates customerKeys.all on success
