  <action>
    1. **src/api/client.ts**: Create Axios instance with:
       - baseURL from EXPO_PUBLIC_API_URL (via Constants.expoConfig.extra.apiUrl)
       - timeout: 15000, Content-Type: application/json
       - Request interceptor: reads accessToken from expo-secure-store, attaches Bearer header
       - Response interceptor: on 401, queues concurrent requests, attempts refresh via /auth/refresh endpoint with refreshToken from SecureStore, on success stores new token pair and retries queued requests, on failure clears SecureStore and navigates to /login via expo-router
       - Use the queue pattern from RESEARCH.md (isRefreshing flag + failedQueue array)

    2. **src/api/auth.api.ts**: Auth API functions using apiClient:
       - login(email, password): POST /auth/login, returns TokenPairDto
       - refresh(refreshToken): POST /auth/refresh, returns TokenPairDto
       - logout(refreshToken): POST /auth/logout
       - getMe(): GET /auth/me, returns UserDto

    3. **src/stores/auth.store.ts**: Zustand store (NOT persisted to AsyncStorage -- tokens in SecureStore):
       - State: user (UserDto | null), isAuthenticated (boolean), isLoading (boolean), biometricEnabled (boolean)
       - Actions: setUser, setAuthenticated, setBiometricEnabled, logout (clears SecureStore + resets state)
       - On init: check SecureStore for existing tokens, validate with getMe(), set user if valid
       - biometricEnabled persisted separately in AsyncStorage (non-sensitive flag)

    4. **src/stores/rental-draft.store.ts**: Zustand with persist middleware:
       - State: step (number), customerId, customerName, vehicleId, vehicleLabel, startDate, endDate, dailyRateNet, rodoConsent (boolean), rodoTimestamp, signatures (string[] -- uploaded URLs)
       - Actions: setStep, updateDraft, clearDraft
       - persist to AsyncStorage with key "rental-draft"
       - initialDraft: { step: 0, signatures: [] }

    5. **src/stores/return-draft.store.ts**: Zustand with persist middleware:
       - State: step (number), rentalId, returnMileage, checklist (Record<string, {damaged: boolean, notes: string}>), notes (string)

       - persist to AsyncStorage with key "return-draft"
