2. **Create PdfService** (`apps/api/src/contracts/pdf/pdf.service.ts`):
   - Injectable class implementing `OnModuleInit` and `OnModuleDestroy`
   - Private `browser: Browser` field (Puppeteer)
   - Private `contractTemplate: HandlebarsTemplateDelegate` and `annexTemplate: HandlebarsTemplateDelegate` — compiled once at init
   - `onModuleInit()`:
     - Launch Puppeteer with `headless: true`, args `['--no-sandbox', '--disable-setuid-sandbox', '--font-render-hinting=none']`
     - Read and compile `contract.hbs` and `annex.hbs` templates using `readFileSync` + `Handlebars.compile`
     - Register Handlebars helpers: `formatDate` (formats ISO date to `DD.MM.YYYY`), `formatDateTime` (formats to `DD.MM.YYYY HH:mm`), `formatMoney` (formats grosze to `X,XX zl`)
   - `onModuleDestroy()`: Close browser
   - `async generateContractPdf(data: ContractFrozenData & { signatures, damageSketch, rodoConsent }): Promise<Buffer>`:
     - Render contract template with data
     - Create new page, setContent with `waitUntil: 'networkidle0'`
     - Wait for fonts: `await page.evaluateHandle('document.fonts.ready')`
     - Generate PDF: `format: 'A4'`, `printBackground: true`, margins 15mm all sides
     - Close page in finally block
     - Return `Buffer.from(pdf)`
   - `async generateAnnexPdf(data: { annexNumber, contractNumber, originalContract, changes }): Promise<Buffer>`:
     - Same pattern using annex template
   - Do NOT use hex colors in templates — use `rgb()` notation (Handlebars `#` conflict)
