2. **Create ContractsService** (`apps/api/src/contracts/contracts.service.ts`):
   Injectable class with constructor dependencies: `PrismaService`, `PdfService`, `StorageService`, `MailService`, `CustomersService`, `ConfigService`

   **Private helper: `generateContractNumber()`**:
   - Query count of contracts created today: `WHERE createdAt >= startOfDay`
   - Format: `KITEK/${year}/${month}${day}/${(count+1).toString().padStart(4, '0')}`

   **Private helper: `buildFrozenData(rental, customer, vehicle, conditions): ContractFrozenData`**:
   - Assemble ContractFrozenData from rental (with vehicle + customer includes) and decrypted customer DTO
   - Company info hardcoded: `{ name: 'KITEK', owner: 'Pawel Romanowski', address: 'ul. Sieradzka 18, 87-100 Torun', phone: '535 766 666 / 602 367 100' }`
   - Customer data from CustomerDto (already decrypted): firstName, lastName, address, pesel, idNumber, idIssuedBy, licenseNumber, licenseCategory, phone, email
   - Vehicle data from rental.vehicle: registration, make, model, year, vin, mileage
   - Rental data: startDate, endDate, dailyRateNet, totalPriceNet, totalPriceGross, vatRate
   - Conditions: depositAmount, dailyRateNet, lateFeeNet

   **Private helper: `generateContentHash(data: ContractFrozenData): string`**:
   - `JSON.stringify(data, Object.keys(data).sort())` for deterministic serialization
   - `crypto.createHash('sha256').update(serialized, 'utf-8').digest('hex')`

   **`async create(dto: CreateContractDto, userId: string): Promise<ContractDto>`**:
   1. Fetch rental with includes (vehicle, customer): `prisma.rental.findUnique({ where: { id: dto.rentalId }, include: { vehicle: true, customer: true } })`
   2. Throw NotFoundException if rental not found
   3. Check no existing SIGNED contract for this rental (allow VOIDED)
   4. Get decrypted customer: `customersService.findOne(rental.customerId)`
   5. Build frozen data with conditions from DTO (depositAmount, lateFeeNet) and rental (dailyRateNet)
   6. Generate content hash from frozen data
   7. Generate contract number
   8. If dto.damageSketchBase64: decode base64, upload to MinIO at `contracts/${rental.id}/damage-sketch.png`
   9. Create contract in Prisma with all fields
   10. Return mapped to ContractDto

   **`async sign(contractId: string, dto: SignContractDto, userId: string, ipAddress: string): Promise<ContractDto>`**:
   1. Fetch contract with signatures
   2. Throw if contract status is SIGNED or VOIDED
   3. Verify content hash matches (re-compute from contractData and compare)
   4. Decode signatureBase64 to Buffer
   5. Upload to MinIO at `contracts/${contract.rentalId}/signatures/${dto.signatureType}.png`
   6. Determine signerRole and signerId from signatureType (customer_* -> role='customer', signerId=null; employee_* -> role='employee', signerId=userId)
   7. Upsert ContractSignature record
   8. Count signatures — if all 4 collected:
      a. Update contract status to SIGNED
      b. Fetch all 4 signature images from MinIO as base64
      c. Fetch damage sketch as base64 if exists
      d. Call `pdfService.generateContractPdf()` with frozen data + signatures + damage sketch + RODO consent
      e. Upload PDF to MinIO at `contracts/${contract.rentalId}/${contract.id}.pdf`
      f. Update contract with pdfKey and pdfGeneratedAt
      g. Get customer email from contract.contractData
      h. If email exists: call `mailService.sendContractEmail()`, update emailSentAt and emailSentTo
   9. Else: update status to PARTIALLY_SIGNED if was DRAFT
   10. Return updated contract

   **`async findOne(id: string): Promise<ContractDto>`**: Fetch with signatures and annexes

   **`async findByRental(rentalId: string): Promise<ContractDto[]>`**: Fetch all contracts for a rental

   **`async createAnnex(rentalId: string, data: { newEndDate: string, createdById: string, newDailyRateNet?: number, newTotalPriceNet?: number }): Promise<ContractAnnexDto>`**:
   1. Find the SIGNED contract for this rental (most recent)
   2. If no signed contract, log warning and return (extension without contract is valid)
   3. Count existing annexes, next number = count + 1
   4. Build changes object: { newEndDate, oldEndDate, newDailyRateNet?, newTotalPriceNet? }
   5. Create ContractAnnex record
   6. Generate annex PDF via `pdfService.generateAnnexPdf()`
   7. Upload to MinIO at `contracts/${rentalId}/annexes/${annexId}.pdf`
   8. Update annex with pdfKey and pdfGeneratedAt
   9. Get customer email from original contract's contractData
   10. If email exists: call `mailService.sendAnnexEmail()`, update emailSentAt
   11. Return annex DTO

   **`toDto(contract: any): ContractDto`**: Map Prisma result to ContractDto with ISO date strings

2. **Create RentalExtendedListener** (`apps/api/src/contracts/listeners/rental-extended.listener.ts`):
   ```typescript
   @Injectable()
   export class RentalExtendedListener {
     private readonly logger = new Logger(RentalExtendedListener.name);
     constructor(private contractsService: ContractsService) {}

     @OnEvent('rental.extended')
     async handleRentalExtended(payload: {
       rentalId: string;
       customerId: string;
       newEndDate: string;
       extendedBy: string;
     }) {
       this.logger.log(`Rental ${payload.rentalId} extended, creating annex`);
       try {
         await this.contractsService.createAnnex(payload.rentalId, {
           newEndDate: payload.newEndDate,
           createdById: payload.extendedBy,
         });
       } catch (error) {
         this.logger.error(`Failed to create annex for rental ${payload.rentalId}: ${error.message}`);
         // Do not throw — annex failure should not block rental extension
       }
     }
   }
   ```
   - Import `OnEvent` from `@nestjs/event-emitter`
   - Error is caught and logged — annex failure must NOT block the rental extension
