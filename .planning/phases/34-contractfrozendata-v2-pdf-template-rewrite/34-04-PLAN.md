**2. Create TipTap editor component** (`tiptap-editor.tsx`):

Props interface:
```typescript
interface TipTapEditorProps {
  content: string;          // Initial HTML content
  onChange: (html: string) => void;  // Called on content change
  editable?: boolean;       // Default true
  className?: string;
}
```

Implementation:
- Use `useEditor` from `@tiptap/react` with StarterKit extension
- StarterKit includes: bold, italic, strike, code, heading, bulletList, orderedList, blockquote, horizontalRule, hardBreak
- Configure headings to allow levels [2, 3] only (no h1 -- too large for terms)
- Render `EditorContent` component from `@tiptap/react`
- Add a simple toolbar above the editor with buttons for:
  - Bold (B), Italic (I), Ordered List (1.), Bullet List, Heading 2, Heading 3
  - Use shadcn Button component with `variant="ghost"` and `size="sm"`
  - Active state: add `bg-muted` class when formatting is active
- Style the editor area with Tailwind: `prose prose-sm max-w-none` for content rendering, `border rounded-md p-3 min-h-[300px]` for the container
- Call `onChange(editor.getHTML())` on every content update via `onUpdate` callback

The component must be a `'use client'` component.

IMPORTANT: Use TipTap v3 API. The key import is `import { useEditor, EditorContent } from '@tiptap/react'` and `import StarterKit from '@tiptap/starter-kit'`.
