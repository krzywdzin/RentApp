**index.tsx (dashboard) -- Greeting fallback (MOBUX-06):**

The current code on line 55 is:
```tsx
const firstName = user?.name?.split(' ')[0] ?? '';
```

This produces an empty string when `user.name` is undefined/empty. Replace with a fallback that uses the email prefix:

```tsx
const firstName = user?.name?.split(' ')[0] || user?.email?.split('@')[0] || '';
```

This way: if name exists, use first name; if name is empty/undefined but email exists, use the part before @ (e.g., "jan.kowalski" from "jan.kowalski@company.pl"); if neither exists, fall back to empty string.
